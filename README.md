# Rechner
### Mit eigener Repl

## Einfache Berechnungen
```
>> 2 + 4
6

>> 5 / 7
0.714285714285714

>> (-4 / 5) ^ 2
-0.8

>> (3 + 2)(3 - 2)
5
```

## Syntax Errors
```
>> 2 + $
-------^ Unbekanntes Zeichen: '$'.

>> 3.4!
------^ Die Fakultät kann nur auf natürliche Zahlen angewendet werden.
```

## Funktionen
```
>> sin(90)
1

>> tan(30)
0.577350269189626

>> cos(235.6)
-0.564967003424938

>> sqrt(2)
1.414213562373095
```

## Variablen
```
>> x = 5
5

>> x / 2
2.5

// Löscht eine Variable
>> del x
```

## Funktionen
```
>> f: x + 5

>> g: x^2 - 1

>> f(4)
9

>> f(5)
24

```
